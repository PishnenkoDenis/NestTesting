## Description

Car booking app repository.

## Installation

```bash
$ npm install
```

## Run seed
You should install postgres
If you are using different username for postgres, change the username in package.json 
```
$ npm run seed
```

## Config environments
Create file `.env` and fill as example (`.example.env`)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
